Dataset Overview

AMOS22 Dataset

Multi-organ segmentation dataset for CT and MRI modalities (Task1 & Task2):

  • ✓ 500 CT scans from multiple centers
  • ✓ 100 MRI scans
  • ✓ 15 abdominal organs annotated
  • ✓ High-quality manual annotations
  • ✓ Multi-center data collection

AMOS-MM Dataset

Multimodal dataset for report generation and visual QA (Task3 & Task4):

  • ✓ Medical images with corresponding reports
  • ✓ Clinical question-answer pairs
  • ✓ Expert-validated annotations
  • ✓ Structured report templates

Evaluation Protocol

AMOS22 Metrics

  • ✓ Dice Similarity Coefficient (DSC)
  • ✓ Normalized Surface Distance (NSD)
  • ✓ Average Symmetric Surface Distance (ASD)

AMOS-MM Metrics

  • ✓ Natural Language Generation metrics
  • ✓ Clinical accuracy assessment
  • ✓ Answer accuracy and relevance
  • ✓ Response completeness

Competition Rules

  1. Teams must register before downloading any dataset
  2. Maximum 5 submissions per day during the validation phase
  3. Final results must be accompanied by a technical report
  4. Code must be made publicly available after the challenge
  5. Teams can participate in any or all tasks
  6. Separate submissions required for each task

Instructions

AMOS22 Data Format

  • ✓ 3D volumes in NIFTI format
  • ✓ Standard orientation
  • ✓ Preprocessed and aligned

AMOS-MM Data Format

  • ✓ JSON format for reports and QA pairs
  • ✓ Structured report templates
  • ✓ Standardized QA format
