Challenge Tasks

CT Segmentation

This task focuses on comprehensive evaluation of segmentation methods across large-scale and diverse CT scans.

Dataset

  • ✓ 500 CT scans from multiple centers
  • ✓ 15 annotated abdominal organs
  • ✓ High-quality manual annotations
  • ✓ Diverse patient conditions

Evaluation Metrics

  • • Dice Similarity Coefficient (DSC)
  • • Normalized Surface Distance (NSD)
  • • Average Symmetric Surface Distance (ASD)

Note: Participants should submit results for all 15 organs to be considered for ranking.

CT-MRI Segmentation

This task extends to both CT and MRI modalities, challenging participants to develop robust cross-modality segmentation methods.

Dataset

  • ✓ 500 CT + 100 MRI scans
  • ✓ Same 15 annotated organs
  • ✓ Consistent annotation protocol
  • ✓ Cross-modality validation

Evaluation Metrics

  • • Same metrics as Task 1
  • • Additional cross-modality consistency measures
  • • Modality-specific performance analysis

Note: Methods will be evaluated on their ability to handle both modalities effectively.

Report Generation

This task focuses on generating accurate and comprehensive medical reports from medical images.

Dataset

  • ✓ Medical images with corresponding reports
  • ✓ Structured report templates
  • ✓ Expert-annotated findings
  • ✓ Multi-modal information

Evaluation Metrics

  • • Natural Language Generation metrics
  • • Clinical accuracy assessment
  • • Report completeness evaluation

Note: Reports will be evaluated for both linguistic quality and clinical accuracy.

Visual Question Answering

This task challenges participants to develop systems that can answer clinical questions about medical images.

Dataset

  • ✓ Medical image-question pairs
  • ✓ Diverse question types
  • ✓ Expert-validated answers
  • ✓ Clinical context information

Evaluation Metrics

  • • Answer accuracy
  • • Clinical relevance
  • • Response completeness

Note: Answers will be evaluated for both accuracy and clinical relevance.
